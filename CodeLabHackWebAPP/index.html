About us

Code Lab is a youth led Tech Ed program created by Rusty and River fields with the goal of teaching one-thousand students in Bed-Stuy coding and robotics this year through Hackathons, in-school coding events, weekend tech programs, and more.

Saturday, April 28th, we are hosting a one-day Hackathon at Crispus Attucks Midlle School in Bed-Stuy, New York. From 9am - 4pm students will learn what is coding, how to hack the web using a really interesting tool called inspect element, HTML, and at the end of the day will ceperate into teams and collaboratively create their own website to showcase among their pears and Code Lab staff
