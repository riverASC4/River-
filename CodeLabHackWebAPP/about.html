About us

Code Lab is a youth led Tech Ed Program created by Rusty and River Fields with the goal of teaching one-thousand students in Bed-Stuy this year coding and robotics through Hackathons, in-school coding events, weekend tech programs, and more.

Saturday, April 28th, we are hosting a one-day Hackathon at Crispus Attucks Midlle School in Bed-Stuy, Brooklyn. From 9am - 4pm students will learn what is coding and computer programming, how to hack the web using a really interesting tool called inspect element which will allow them to change different features of any web site on their web browser, HTML, and at the end of the day will seperate into teams and collaboratively create their own website to showcase among their peers and Code Lab staff
